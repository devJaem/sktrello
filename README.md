# SKTrello

- 트렐로와 비슷한 칸반보드 기반의 협업툴 서비스 구현.
- Nest.js, TypeORM을 이용하여 구현한다.

## 👋 SKTrello 팀 소개

- **SKTrello**는 저희 팀명과 트렐로를 합성한 간단한 이름입니다.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
